Python

Introduction

Python is a high-level, general purpose programming language designed to be beginner friendly and readable.
It's also garbage collected, very practical not only for datascience and vast majority of tasks with computers but also real life situatuions, And python's a very popular programming language, in fact it is the one of the most popular language out there.
It was created for Server-Side applications, machine learning and the best of all which is automation.
It has gained its popularity from Simplicity and readability for humans. It also supports wide variety of use cases.
The wide variety of use cases comes from Pythons massive library filled with different modules for many cases.
For example, the module 'os' can do system operations like getting information about the system and controlling files and directories inside your computer.

History

Python was designed and developed by Guido Van Rossum at Netherlands Centrum Wiskunde & Informitica in december of the year 1989 but first appeared in february of the year 1991 with the version Python 0.9.0 .
It's said that the name 'Python' comes from the comedy show Monty Python's Flying Circus
Python 2.0 was released in october 2000 with new important features list comprehensions, cycle detecting garbage collection, reference counting and the last of all, Unicode support. The update was really important for modern python programs and operations.
In 2008 Python 3.0 was released with changed syntax and new semantics. With the Python 3.0 release it quickly grew very popular in 2010 and eventually became rivals with java and javascript, and since then it has been one of the most popular (also the easiest) programming language.

Features and philosophy

Python's a multi paradigm language. Python fully supports OOP(Object oriented programming) and structured programming. Also many of their features support functional programming, including aspect oriented programming, metaprogramming and metaobjects.
Python uses dynamic typing, a combination of reference counting and a cycle detecting garbage for managing memory. It also uses dynamic name resolution or late binding, which binds methods and variables during program execution.
Pythons design offers support for functionality while programming in the Lisp tradition. It has filter, map and reduce functions; list comprehensions, dictionaries, sets and generator expressions.
To get deeper into the philosophy(or core philosophy), it is summarized in the Zen of python.

Programming

Python has an Indentation rule which if you make a statement or a function you have to indent the code inside the statement as a whitespace, otherwise it is going to run as the program starts or inside another statement.
Indentation is necessary for also creating classes which we'll talk about later
Python has 9 datatypes which are:

Strings (str in Python)
Integers (int in Python)
Floats (float in Python)
Arrays (list in Python)
Sets (set in Python)
Maps (dict (which is shorthand for dictionary) in Python)
Tuples (tuple in python)

Python has 35-36 keywords and statements but the most important of them are:

  • if statement to make a conditional statement.
  • elif statement makes another condition, when an if statements expression is not true. You never type it before if statement.
  • else statements runs when an if or elif statements condition is not true.

Example of if-elif-else statements:

if (Condition1):
    (Expression)
elif Condition2:
    (Expression)
else
    (Expression)

Note: you can put multiple elif statements under one if statement

  • for loops are for looping (or iterating) over iterable objects like lists and dicts.
  • while loops is repeating an expression while an condition is true

Example of a while loop:

while (Condition):
    (Expression)

  • try statements can handle errors with the except keyword while the code is running. There is also the finally keyword that always runs even if an error occurs.

    • There is a bunch of different errors that can occur, for example ValueError or TypeError. So you can handle them by using the except keyword. Here is an example:

    try:
        (Expression1)
    except ValueError:
        (Expression2 in case of a ValueError)
    finally:
        (Expression3 that always runs no matter what.)

    • raise keyword can assert an error like ValueError
    • class keyword is used to create classes that might be necessary in some programs. You can define classes using Object-Oriented-Programming.
    • def keyword is used to define functions.
    • return keyword is used in functions to return a value when the function is called.
    • import keyword can use other Python programs classes or functions by typing import and then the Python programs name (if the program is in the same package). You can also import modules from the massive library of python.
